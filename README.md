# Legal-Ease Android

Legal-Ease is a smart legal assistance application designed to bridge the gap between lawyers and clients. This repository contains the Android application built using Flutter.

## Features

- **Lawyer Dashboard:** Manage active cases, pending requests, and legal documents.
- **Client Management:** Real-time chat synchronization with clients.
- **Vakalatnama Management:** Generate and manage Vakalatnama forms with digital signatures.
- **Hearing Schedules:** Set and notify clients about upcoming hearing dates and details.
- **Document Vault:** Securely upload and download legal documents.
- **Admin Panel:** Verification system for lawyers.

## Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- [Firebase Account](https://firebase.google.com/) (configured for the project)
- Android Studio / VS Code

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/labizareen-collab/Legal-Ease-Android.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Legal-Ease-Android
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```

## Technologies Used

- **Framework:** Flutter
- **Language:** Dart
- **Backend:** Firebase (Firestore, Auth, Storage)
- **State Management:** Provider / Streams
